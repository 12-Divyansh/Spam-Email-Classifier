# Spam Email Classifier

## Overview
This project uses Machine Learning to classify SMS or Email messages as Spam or Not Spam.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- Streamlit

## Machine Learning Algorithm
- CountVectorizer
- Multinomial Naive Bayes

## Accuracy
98%

## Features
- Spam Detection
- Simple User Interface
- Fast Prediction

## How to Run

pip install -r requirements.txt

streamlit run app.py
